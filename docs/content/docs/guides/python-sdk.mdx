# Python SDK Guide

Idiomatic Python bindings for Crawlingo's Rust core, supporting context managers, hooks, Pandas integration, and fluent APIs.

## Installation

```bash
pip install crawlingo
```

Requires Python 3.9+. Pre-compiled wheels for Windows, macOS, Linux (x86_64, aarch64).

## Key Differences from Node.js

| Feature | Python | Node.js |
|---------|--------|---------|
| Session factory methods | `.page()`, `.dataset()`, `.crawl()`, `.watch()` | Pass Session to constructors |
| Page creation | `Page(url, session=s)` | `await Page.create(url, {session})` |
| Element navigation | `.parent()`, `.children()`, `.next()`, `.prev()`, `.siblings()` | Not available |
| ElementCollection filter/map | `.filter(fn)`, `.map(fn)` | Not available |
| Hooks | `.before_fetch()`, `.after_fetch()`, `.before_parse()`, `.after_extract()` | Not available |
| Context manager | `with Session() as s:` | N/A |
| Pandas integration | `.df()` on DatasetResult, CrawlResults | Not available |
| CLI | `crawlingo shell`, `extract`, `mcp` | N/A |

## Session

```python
from crawlingo import Session

session = Session()
session.auto_match(True)
session.headers({"User-Agent": "MyBot/1.0"})
session.rate_limit(3.0)

# Context manager
with Session() as s:
    page = s.page("https://example.com")

# Factory methods
page = session.page("https://example.com")
dataset = session.dataset("https://example.com/item/1")
crawl = session.crawl("https://example.com/blog")
watch = session.watch("https://example.com/product/1")
```

## Page

```python
from crawlingo import Page

# Direct construction
page = Page("https://example.com")

# With session
page = Page("https://example.com", session=session)

# Via session
page = session.page("https://example.com")

# Properties
print(page.url)     # str
print(page.status)  # int

# Content
print(page.html())   # Raw HTML
print(page.title())  # <title> text

# Selectors
page.css("div.product")
page.xpath("//div[@class='price']")
page.find_text("Add to cart")
page.after_text("Price:")
page.before_text("Description:")
page.regex(r"\$\d+\.\d{2}")
```

## Element

```python
el = page.css("div.product").first()

el.text()       # Recursive text
el.html()       # HTML snippet
el.attr("href") # Attribute value
el.attrs()      # All attributes

# Navigation
el.parent()       # Parent Element or None
el.children()     # ElementCollection of children
el.next()         # Next sibling or None
el.prev()         # Previous sibling or None
el.siblings()     # ElementCollection of siblings
```

## ElementCollection

```python
col = page.css("div.product")

col.first()           # First Element
col.last()            # Last Element
col.nth(2)            # Element at index 2
col[0]                # Index access
col.text()            # Combined text
col.texts()           # List of texts
col.attr("href")      # First element's attr
col.attrs()           # First element's attrs
len(col)              # Count
for el in col:        # Iteration
col.filter(lambda e: "price" in e.text())  # Filter
col.map(lambda e: e.attr("href"))          # Map
```

## Dataset

```python
from crawlingo import Dataset

dataset = Dataset("https://shop.example.com/item/42")
dataset.field("name", "h1.title")
dataset.field("price", "span.price", default="$0.00")
dataset.field("description", "meta[name='description']")
dataset.field("sku", ".sku-code", selector_type="xpath")

def clean_price(val):
    return val.replace("$", "").strip()

dataset.field("price_clean", "span.price", transform=clean_price)

# Sync
result = dataset.build()

# Async
import asyncio
result = asyncio.run(dataset.build_async())

# Export
result.to_json("output.json")
result.to_csv("output.csv")
result.to_parquet("output.parquet")

# Access
print(result.to_dict())
print(result["name"])
print(result.df())  # Pandas DataFrame
```

## Crawl

```python
from crawlingo import Crawl

crawl = Crawl("https://blog.example.com")
crawl.follow("a.next-page")
crawl.limit(50)
crawl.depth(3)
crawl.concurrency(5)
crawl.delay(0.5)
crawl.field("title", "h1.entry-title")
crawl.field("author", "span.author-name", default="Anonymous")

# Webhook for real-time delivery
crawl.webhook("https://my-api.com/webhook")

# Run
results = crawl.build()

# Schedule recurring crawl
crawl.schedule(3600)

# Export
results.to_json("posts.json")
results.to_csv("posts.csv")
results.to_parquet("posts.parquet")
results.df()  # Pandas DataFrame

# Iterate
for result in results:
    print(result["title"])
```

## Watch

```python
from crawlingo import Watch

watch = Watch("https://example.com/product/1")
watch.field("price", "span.price")
watch.field("stock", ".stock-badge")
watch.interval(300)  # every 5 minutes

# Callbacks
watch.on_change(lambda ev: print(f"Change: {ev.field}"))
watch.on_price_change(lambda ev: print(f"Price: {ev.old_value} -> {ev.new_value}"))
watch.on_stock_change(lambda ev: print(f"Stock changed!"))
watch.on_element_added(lambda ev: print(f"New element: {ev.field}"))
watch.on_element_removed(lambda ev: print(f"Element gone: {ev.field}"))

# Run (blocks)
watch.run()

# Or async
import asyncio
asyncio.run(watch.run_async())
```

## Hooks

```python
from crawlingo import Page, hooks

page = Page("https://example.com")
page.before_fetch(hooks.log_request)
page.after_fetch(hooks.log_response)
page.before_parse(lambda html: html.replace("old", "new"))
page.after_extract(lambda val: val.strip())

# Built-in hooks
from crawlingo.hooks import strip_whitespace, uppercase, lowercase, log_request, log_response
```

## Error Handling

```python
from crawlingo import (
    CrawlingoError,
    FetchError,
    ParseError,
    SelectorError,
    AutoMatchFailed,
    TimeoutError,
    RateLimitError,
    ChangeDetectionError,
    ExportError,
    DnsError,
    FingerprintStoreError,
)

try:
    page = Page("https://example.com")
    page.title()
except FetchError as e:
    print(f"Network error: {e}")
except TimeoutError as e:
    print(f"Timeout: {e}")
except SelectorError as e:
    print(f"Bad selector: {e}")
except CrawlingoError as e:
    print(f"Generic error: {e}")
```

## CLI

```bash
# Interactive shell
crawlingo shell
crawlingo shell https://example.com

# One-shot extraction
crawlingo extract https://example.com --css "h1.title" --json

# MCP server
crawlingo mcp --host 0.0.0.0 --port 8000
```
