# Third-Party Integrations

Connect your scraping sessions directly to Python analysis pipelines, big data storage files, or AI Agent protocols.

---

## Pandas DataFrames

Export scraped datasets directly into Pandas DataFrames for machine learning or analytical pipelines.

<div className="grid grid-cols-1 gap-4 my-6">
  <HoverBorderCard className="p-6">
    <Tabs items={['Python', 'Node.js']}>
      <Tab value="Python">
        ```python
        from crawlingo import Session

        session = Session()
        dataset = session.dataset("https://example.com/products")
        dataset.field("name", ".product-name")

        result = dataset.build()
        df = result.df()  # Returns standard pandas.DataFrame
        print(df.head())
        ```
      </Tab>
      <Tab value="Node.js">
        ```typescript
        import { Dataset } from 'crawlingo';

        const dataset = new Dataset("https://example.com/products");
        dataset.field("name", ".product-name");

        const result = await dataset.build();
        // Use result.toDict() for data analysis
        console.log(result.toDict());
        ```
      </Tab>
    </Tabs>
  </HoverBorderCard>
</div>

## Model Context Protocol (MCP)

Built-in Model Context Protocol server lets LLM agents crawl URLs, inspect headers, and build dynamic schemas.

<HoverBorderCard className="p-6 my-4">
  <p className="text-xs text-gray-500 dark:text-gray-400 mb-4">Built-in Model Context Protocol server lets LLM agents crawl URLs, inspect headers, and build dynamic schemas.</p>
  ```bash
  # Launch the MCP server from command line
  crawlingo mcp --host 127.0.0.1 --port 8000

  # Available tools:
  # - fetch_page: Returns URL body text & headers.
  # - extract_data: Executes one-shot structural extraction.
  ```
</HoverBorderCard>

## IP Proxy Rotation & Pools

Set up static lists of rotating proxy servers or integrate directly with proxy API providers.

<div className="grid grid-cols-1 gap-4 my-6">
  <HoverBorderCard className="p-6">
    <Tabs items={['Python', 'Node.js']}>
      <Tab value="Python">
        ```python
        session = Session()
        session.proxy_pool([
            "http://proxy1.example.com:8080",
            "http://proxy2.example.com:8080"
        ])
        session.proxy_provider("https://api.proxydomain.com/v1/list?key=auth")
        session.rate_limit(2.0)
        ```
      </Tab>
      <Tab value="Node.js">
        ```typescript
        const session = new Session();
        session.proxyPool([
          "http://proxy1.example.com:8080",
          "http://proxy2.example.com:8080"
        ]);
        session.proxyProvider("https://api.proxydomain.com/v1/list?key=auth");
        session.rateLimit(2.0);
        ```
      </Tab>
    </Tabs>
  </HoverBorderCard>
</div>

## Real-time Webhook Delivery

Stream extracted page records to external webhooks instantly, avoiding huge local memory overhead.

<div className="grid grid-cols-1 gap-4 my-6">
  <HoverBorderCard className="p-6">
    <Tabs items={['Python', 'Node.js']}>
      <Tab value="Python">
        ```python
        session = Session()
        crawl = session.crawl("https://example.com/products")
        crawl.follow("a.next-page")
        crawl.field("title", "h1")
        crawl.webhook("https://api.yourservice.com/webhooks/crawl")
        crawl.build()
        ```
      </Tab>
      <Tab value="Node.js">
        ```typescript
        const session = new Session();
        const crawl = new Crawl("https://example.com/products", session);
        crawl.follow("a.next-page");
        crawl.field("title", "h1");
        crawl.webhook("https://api.yourservice.com/webhooks/crawl");
        await crawl.run();
        ```
      </Tab>
    </Tabs>
  </HoverBorderCard>
</div>
