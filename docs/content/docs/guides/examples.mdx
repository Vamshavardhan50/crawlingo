### 61. Monitor Product Price Drop

```python
watch = Watch("https://shop.example.com/product/123")
watch.field("name", "h1.product-title")
watch.field("price", "span.price-final")
watch.interval(3600)  # Check every hour

def price_alert(ev):
    try:
        old = float(ev.old_value.replace("$", ""))
        new = float(ev.new_value.replace("$", ""))
        if new < old:
            print(f"🔥 PRICE DROP! {ev.new_value} (was {ev.old_value})")
    except ValueError:
        pass

watch.on_price_change(price_alert)
watch.run()
```

## Edge Cases

### 96. Empty Page

```python
page = Page("https://example.com/empty")
result = page.css("h1")
print(result.text())  # "" (empty string)
print(len(result))    # 0
```

### 97. Page with No Title

```python
page = Page("https://example.com/notitle")
print(page.title())  # "" (empty string, no error)
```

### 98. Invalid Regex Pattern

```python
try:
    page.regex(r"[invalid")  # Unclosed bracket
except SelectorError as e:
    print(f"Bad regex: {e}")
```

### 99. Auto-Match with No Fingerprint

```python
page = Page("https://example.com", auto_match=True)
# First query succeeds, creates fingerprint
result = page.css("span.price")
print(result.text())

# If selector still works, auto-match not triggered
# If it fails, auto-match tries to recover
```

### 100. Watch with Rapid Polling

```python
watch = Watch("https://example.com")
watch.field("price", ".price")
watch.interval(1)  # Every second — aggressive!
watch.on_change(lambda ev: print(f"Change: {ev.new_value}"))
# watch.run() — runs forever
```

### 101. Very Large Page

```python
page = Page("https://large.example.com")
print(len(page.html()))  # Full HTML string
# DOM tree is parsed streaming, memory efficient
```

### 102. Page Behind Login

```python
session = Session()
session.cookies({"session": "valid-token"})
page = session.page("https://dashboard.example.com")
```

### 103. Page with Redirect

```python
page = Page("https://httpbin.org/redirect-to?url=https://example.com")
print(page.url)    # Final URL after redirects
print(page.status)  # 200
```
