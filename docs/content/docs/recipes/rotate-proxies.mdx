# Rotate Proxies

## Problem

Target servers block or rate-limit your IP address. You need to distribute requests across multiple proxy servers.

## Solution

Use `proxy_pool()` for a static list or `proxy_provider()` for dynamic proxy discovery.

<Tabs items={['Python', 'Node.js']}>
  <Tab value="Python">

    ```python
    from crawlingo import Session

    session = Session()

    # Static pool — round-robin rotation
    session.proxy_pool([
        "http://user:pass@us-proxy1:8080",
        "http://user:pass@eu-proxy2:8080",
        "http://user:pass@asia-proxy3:8080",
        "socks5://user:pass@proxy4:1080",
    ])

    # Dynamic provider — fetched at runtime
    session.proxy_provider("https://api.proxy-service.com/v1/list?key=YOUR_KEY")
    ```

  </Tab>
  <Tab value="Node.js">

    ```typescript
    import { Session } from 'crawlingo';

    const session = new Session();

    session.proxyPool([
      "http://user:pass@us-proxy1:8080",
      "http://user:pass@eu-proxy2:8080",
    ]);

    session.proxyProvider("https://api.proxy-service.com/v1/list?key=YOUR_KEY");
    ```

  </Tab>
</Tabs>

## Best Practices

- Combine proxy rotation with stealth fetcher mode
- Use SOCKS5 proxies for UDP support
- Validate proxies before adding to pool
- Monitor proxy response times and remove slow ones
- Set rate limits conservative enough for the slowest proxy
