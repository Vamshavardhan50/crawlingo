## Session

Shared configuration container for cookies, headers, proxy, rate limits, and auto-match settings. All configuration methods return `self` for fluent chaining.

### Parameters

| Method | Type | Default | Description |
|--------|------|---------|-------------|
| `.headers(dict)` | `dict[str,str]` | `{}` | HTTP headers applied to every request |
| `.cookies(dict)` | `dict[str,str]` | `{}` | Cookie jar shared across requests |
| `.proxy(url)` | `str` | `None` | Proxy URL (http, https, socks5) |
| `.rate_limit(rps)` | `float` | `0.0` | Max requests per second per host |
| `.auto_match(bool)` | `bool` | `False` | Enable self-healing selector recovery globally |
| `.timeout(secs)` | `int` | `30` | Request timeout in seconds |
| `.fingerprint_path(path)` | `str` | `".crawlingo"` | Directory for fingerprint store database (sled) |
| `.fetcher_tier(tier)` | `str` | `"standard"` | `"standard"` or `"stealthy"` |
| `.browser_profile(profile)` | `str` | `None` | `"chrome"`, `"firefox"`, or `"safari"` |
| `.auto_match_weights(dict)` | `dict[str,float]` | `{}` | Custom similarity weights for auto-match |
| `.proxy_pool(list)` | `list[str]` | `[]` | Rotating proxy list (round-robin) |
| `.proxy_provider(url)` | `str` | `None` | API endpoint that returns a proxy list |

### Factory Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `.page(url)` | `Page` | Create Page inheriting session config |
| `.dataset(url)` | `Dataset` | Create Dataset inheriting session config |
| `.crawl(url)` | `Crawl` | Create Crawl inheriting session config |
| `.watch(url)` | `Watch` | Create Watch inheriting session config |

## Page

Fetched web page with a parsed immutable DOM tree and selector query methods.

### Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `url` | `str` | — | Target URL to fetch |
| `auto_match` | `bool` | `False` | Enable self-healing selector fallback |
| `timeout` | `int` | `30` | Request timeout in seconds |
| `retries` | `int` | `3` | Retry count on transient failures |
| `headers` | `dict[str,str]` | `None` | Custom HTTP headers |
| `cookies` | `dict[str,str]` | `None` | Custom cookies |
| `proxy` | `str` | `None` | Proxy URL |
| `session` | `Session` | `None` | Session to inherit settings from |

**Session merge order:** Session values act as defaults. Explicit Page constructor parameters override session values.

### Properties

| Property | Type | Description |
|----------|------|-------------|
| `.url` | `str` | The resolved target URL |
| `.status` | `int` | HTTP response status code |

### Methods

| Method | Returns | Description |
|--------|---------|-------------|
| `.html()` | `str` | Raw HTML source (O(1) access) |
| `.title()` | `str` | `<title>` tag text content |
| `.css(sel)` | `ElementCollection` | CSS selector query (cached) |
| `.xpath(query)` | `ElementCollection` | XPath expression query |
| `.find_text(text)` | `ElementCollection` | SIMD text anchor search |
| `.after_text(text)` | `ElementCollection` | Sibling after text anchor |
| `.before_text(text)` | `ElementCollection` | Sibling before text anchor |
| `.regex(pattern)` | `ElementCollection` | Regex pattern match on text |

### Hooks

| Method | Returns | Description |
|--------|---------|-------------|
| `.before_fetch(fn)` | `Page` | Hook before HTTP fetch |
| `.after_fetch(fn)` | `Page` | Hook after HTTP fetch |
| `.before_parse(fn)` | `Page` | Hook before HTML parsing |
| `.after_extract(fn)` | `Page` | Hook after value extraction |

## Dataset

### Parameters

| Constructor Parameter | Type | Default | Description |
|----------------------|------|---------|-------------|
| `url` | `str` | — | Target URL to fetch and extract from |
| `session` | `Session` | `None` | Optional session (creates default if omitted) |

### `.field()` Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `name` | `str` | — | Output field name (used as key in result dict) |
| `selector` | `str` | — | Selector expression |
| `selector_type` | `str` | `"css"` | `"css"`, `"xpath"`, `"regex"`, `"text"`, `"after_text"`, `"before_text"` |
| `transform` | `Callable` | `None` | Python callable `fn(str) -> str` applied to extracted value |
| `default` | `str` | `None` | Fallback value if selector yields nothing |

### Methods

| Method | Returns | Python | Node.js |
|--------|---------|--------|---------|
| `.field(name, sel, ...)` | `Dataset` | ✅ | ✅ |
| `.auto_match(bool)` | `Dataset` | ✅ | ✅ |
| `.timeout(secs)` | `Dataset` | ✅ | ✅ |
| `.headers(dict)` | `Dataset` | ✅ | ✅ |
| `.build()` | `DatasetResult` | ✅ (sync) | ✅ (async) |
| `.build_async()` | `DatasetResult` | ✅ | ❌ |
| `.extractStructured(page)` | `Record[]` | ❌ | ✅ |
| `.buildStructured()` | `Promise<Record[]>` | ❌ | ✅ |
| `static saveJson(records, path)` | `void` | ❌ | ✅ |
| `static saveCsv(records, path)` | `void` | ❌ | ✅ |

### `.build()` Algorithm

1. Read session config (headers, cookies, proxy, rate limit, timeout, fetcher tier) at call time
2. Build `FetchRequest` with session settings
3. Dispatch via `FetchManager` with rate limiting and retries
4. Parse HTML response via `HtmlParser::parse()` into `Page`
5. For each field:
   a. Build `SelectorQuery` from `selector_type` + `selector`
   b. Run against DOM tree
   c. If empty AND auto-match enabled AND CSS: run auto-matcher recovery
   d. Extract combined text from matched nodes
   e. Apply transform (Python only)
6. Return `DatasetResult`

## Crawl

### Parameters

| Constructor Parameter | Type | Default | Description |
|----------------------|------|---------|-------------|
| `start_url` | `str` | — | Starting URL |
| `session` | `Session` | `None` | Optional session |

### Configuration Methods

| Method | Default | Description | Python | Node.js |
|--------|---------|-------------|--------|---------|
| `.follow(sel)` | `""` | CSS selector for links to follow | ✅ | ✅ |
| `.limit(n)` | `10` | Maximum pages to crawl | ✅ | ✅ |
| `.depth(n)` | `3` | Maximum link hop depth | ✅ | ✅ |
| `.field(...)` | — | Field definition | ✅ | ✅ |
| `.concurrency(n)` | `2` | Concurrent fetch workers | ✅ | ✅ |
| `.delay(secs)` | `0.0` | Politeness delay between fetches | ✅ | ✅ |
| `.webhook(url)` | `None` | Webhook URL for real-time JSON delivery | ✅ | ✅ |
| `.schedule(secs)` | — | Recurring crawl in background thread | ✅ | ✅ |
| `.build()` / `.run()` | — | Execute crawl | ✅ (`.build()`) | ✅ (`.run()`) |

### Algorithm

1. Start URL enqueued at depth 0
2. Spawn `concurrency` workers as a `JoinSet`
3. Each worker pops URL from shared queue, checks visited set, fetches, parses, extracts fields
4. Links matching `follow` selector from depth d are enqueued at depth d+1
5. Worker exits when queue is empty or limit is reached
6. Returns `CrawlResults` (Python) or `DatasetResult[]` (Node.js)

## CLI

```bash
crawlingo shell [url]
crawlingo extract <url> [--css] [--xpath] [--text] [--regex] [--auto-match] [--json]
crawlingo mcp [--host HOST] [--port PORT]
```

### `shell`

Start interactive Python REPL with pre-imported classes.

| Argument | Type | Description |
|----------|------|-------------|
| `url` (optional) | `str` | URL to pre-fetch into a `page` variable |

Pre-imported: `Page`, `Dataset`, `Crawl`, `Watch`, `Session`.

### `extract`

One-shot extraction from a URL.

| Flag | Type | Description |
|------|------|-------------|
| `<url>` | `str` | Target URL |
| `--css` | `str` | CSS selector |
| `--xpath` | `str` | XPath expression |
| `--text` | `str` | Text anchor |
| `--regex` | `str` | Regex pattern |
| `--auto-match` | flag | Enable auto-match |
| `--json` | flag | JSON output |

### `mcp`

Start MCP SSE server.

| Flag | Default | Description |
|------|---------|-------------|
| `--host` | `127.0.0.1` | Bind address |
| `--port` | `8000` | TCP port |

Protocol: SSE at `/sse`, POST to `/message`.

## Exceptions

| Exception | Description | When Raised |
|-----------|-------------|-------------|
| `CrawlingoError` | Base exception | Generic errors |
| `FetchError` | Network failure | Connection refused, DNS failure, proxy error, SSL error |
| `ParseError` | HTML parse failure | Invalid HTML, encoding issues |
| `SelectorError` | Invalid selector | Invalid CSS/XPath/regex syntax |
| `AutoMatchFailed` | Auto-match recovery failed | No DOM element above 0.5 similarity threshold |
| `TimeoutError` | Request timeout | Response exceeded configured timeout |
| `RateLimitError` | Host rate limit | Token bucket exhausted (not currently raised — rate limiter blocks instead) |
| `ChangeDetectionError` | Change detection failure | Internal processing error in `detect_changes` |
| `ExportError` | Export failure | File write error, CSV/Parquet serialization error |
| `DnsError` | DNS resolution failure | Host not found, DNS server unreachable |
| `FingerprintStoreError` | Fingerprint store failure | Sled database read/write error |
