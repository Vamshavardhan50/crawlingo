# Your First Crawl

This guide walks through building a production-ready multi-page crawler that discovers pages via link following and extracts structured fields from each page.

## What You Will Build

A crawler that:
1. Starts from a blog index page
2. Follows "next page" links
3. Extracts post titles and dates from 50 pages
4. Exports results to CSV

## Step 1: Create a Session

A `Session` holds shared configuration (rate limits, headers, proxy settings). Always start here for production crawls.

<Tabs items={['Python', 'Node.js']}>
  <Tab value="Python">

    ```python
    from crawlingo import Session

    session = Session()
    session.rate_limit(2.0)  # 2 requests per second max
    session.timeout(30)
    ```

  </Tab>
  <Tab value="Node.js">

    ```typescript
    import { Session } from 'crawlingo';

    const session = new Session();
    session.rateLimit(2.0);
    session.timeout(30);
    ```

  </Tab>
</Tabs>

## Step 2: Create the Crawl

<Tabs items={['Python', 'Node.js']}>
  <Tab value="Python">

  </Tab>
  <Tab value="Node.js">

## Step 3: Configure Link Following

Tell the crawler which links to follow. This CSS selector targets "next page" links:

<Tabs items={['Python', 'Node.js']}>
  <Tab value="Python">

    ```python

    ```

  </Tab>
  <Tab value="Node.js">


  </Tab>

## Step 4: Set Limits

Control how many pages to crawl and how deep to go:

<Tabs items={['Python', 'Node.js']}>
  <Tab value="Python">

    ```python
    crawl.limit(50)   # Stop after 50 pages
    crawl.depth(3)    # Maximum 3 hops from start
    ```

  </Tab>
  <Tab value="Node.js">

    ```typescript

    crawl.depth(3);
    ```

  </Tab>
</Tabs>

## Full Example

Here is the complete script:

<Tabs items={['Python', 'Node.js']}>
  <Tab value="Python">

    ```python
    from crawlingo import Session

    session = Session()
    session.rate_limit(2.0)

    crawl = session.crawl("https://blog.example.com")
    crawl.follow("a.next-page")
    crawl.limit(50)
    crawl.depth(3)
    crawl.concurrency(5)
    crawl.field("title", "h1.entry-title")
    crawl.field("date", "time.published")
    crawl.field("author", ".author-name")

    results = crawl.build()
    results.to_csv("blog_posts.csv")
    print(f"Done — {len(results)} pages crawled")
    ```

  </Tab>
  <Tab value="Node.js">

    ```typescript
    import { Session, Crawl } from 'crawlingo';

    const session = new Session();
    session.rateLimit(2.0);

    const crawl = new Crawl("https://blog.example.com", session);
    crawl.follow("a.next-page");
    crawl.limit(50);
    crawl.depth(3);
    crawl.concurrency(5);
    crawl.field("title", "h1.entry-title");
    crawl.field("date", "time.published");
    crawl.field("author", ".author-name");

    const results = await crawl.run();
    console.log(`Done — ${results.length} pages crawled`);
    ```

  </Tab>
</Tabs>
